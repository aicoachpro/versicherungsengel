# /research — Recherche

Recherchiert technische Fragen und liefert quellengestützte Ergebnisse.

## Workflow

1. **Frage verstehen** — Was genau soll recherchiert werden?
2. **Quellen durchsuchen** — Web, Dokumentation, Code-Beispiele
3. **Ergebnisse strukturieren** — Pro/Contra, Empfehlung, Quellen
4. **Operator präsentieren** — Zusammenfassung + nächste Schritte

## Einsatz

- API-Evaluierung (Kosten, Limits, Alternativen)
- Framework-Vergleiche
- Best Practices für spezifische Patterns
- Sicherheitsfragen
